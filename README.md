# Discord Image Classification Bot

A Discord bot that detects and classifies images uploaded in a server using a trained machine learning model.  
It provides both classification results and metadata about the uploaded images.

---

## 🚀 Features
- `!detect_info` → Saves attachments, runs classification, and returns attachment data
- `!detect` → Saves attachments and returns classification results (label + confidence score).
- Simple integration with any Discord server.
